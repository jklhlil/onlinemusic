# onlinemusic
这是一个使用 vue + vue-router + axios 做的移动端音频播放软件。

本系统主要内容包括：
1.	查询功能，方便用户快速查询某个音频资源。
2.	音频收听功能，用户能流畅的播放想听的音频资源。
3.	订阅功能，对于某些特定的音频资源，用户需要订阅才能试听。
4.	评论功能，用户可以对音频资源进行评论。
5.	收藏功能，用户可以收藏音频资源。
6.	积分功能，用户可以通过活动获取积分并使用积分订阅音频资源。
